"pająka AJAX"

The AJAX Spider add-on integrates in ZAP a crawler of AJAX rich sites called Crawljax. You can use it to identify the pages of the targeted site. You can combine it with the (normal) spider for better results.

The spider is configured using the Options AJAX Spider screen.

Accessed via

     AJAX Spider tab
     Sites tab 'Attack/AJAX Spider site' right click menu item
     Sites tab 'Attack/AJAX Spider in Scope' right click menu item

Zobacz również

     Options AJAX Spider screen for an overview of the AJAX Spider Options
     AJAX Spider tab for an overview of the AJAX Spider Tab
