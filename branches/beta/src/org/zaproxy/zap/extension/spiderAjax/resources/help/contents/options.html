Options AJAX Spider screen

This screen allows you to configure the AJAX Spider options. The AJAX Spider is an add-on for a crawler called Crawljax. The add-on sets up a local proxy in ZAP to talk to Cralwjax. Therefore, you can configure the crawler.

AJAX Spider Crawler Configuration

You can configure the following parameters of Crawljax.

Number of Browser Windows

You can configure the number of windows to be used by AJAX Spider. The more windows, the faster the process will be.

Browser

AJAX Spider relies on an external browser to crawl the targeted site. You can specify which one you want to use: Firefox, Chrome or HtmlUnit.

HtmlUnit is an embedded java based browser included in AJAX Spider.

If you use Firefox or Chrome, you must have it installed in your system.

Chrome requires a specific package to run, called Chrome Driver.
If you want to use Chrome, you should install it. For more information refer to http://code.google.com/p/chromedriver/downloads/list

Select ChromeDriver

Allows to select the location of the Chrome Driver.

Crawl in depth

If you enable this option, more HTML tags will be analyzed to find new links. Following tags will be analyzed: a, button, td, span, div, tr, ol, li, radio, form, select, input, option, img, p, abbr, address, area, article, aside, audio, canvas, details, footer, header, label, nav, section, summary, table, textarea, th, ul, video. This usually improves overall results but slows down the process.

Maximum crawl depth

The maximum depth that the crawler can reach. The default is 10. Unlimited depth is zero.

Maximum crawl states

The maximum number of states that the crawler should crawl. The default is unlimited. Unlimited crawl states is zero.

Maximum duration

The maximum time that the crawler should run. Default is one hour. Unlimited running time is zero. Crawler will stop if one of other limiting conditions is reached.

Event wait time

The time to wait after an event is fired. Default is 1000 milliseconds.

Reload wait time

The time to wait after URL is loaded. Default is 1000 milliseconds.

Click elements once

Sets the crawler to interact with any element only once. Default is true.
