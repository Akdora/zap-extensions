Zest Assertion: Regex

This assertion tests whether the specified regex is present in the response body.

Attributes

Name Format Description
regex String The regular expression to test for
inverse String If false then the assertion passes if the regex is present, otherwise only if it is absent
index Integer The index of the statement in the script
elementType String Always ZestAssertBodyRegex

JSON

An basic example: 
{
  "regex": "You have logged in",
  "reverse": false,
  "index": 2,
  "elementType": "ZestAssertBodyRegex"
}
